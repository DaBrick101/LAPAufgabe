In diesem Projekt geht es darum Einträge einer DB zu veranschaulichen, editieren, filtern, erstellen und löschen

Es wurd eine simple DB verwendet mit einem Table mit Personen.
Es gibt folgende attribute:
    - ID
    - Name
    - Prename
    - Email
    - Date of Birth

Um das Projekt umzubauen müssen nur die DB Connection, Text auf der Seite und die DB abfragen bearbeitet werden und eventuell müssen die Seiten umgebaut werden.

Das Projekt ist gerade im Aufbau und noch nicht fertig.
